## Overview

**BlackRoad Sandbox** provides isolated development and testing environments for the [BlackRoad OS](https://blackroad.io) platform — an enterprise AI operating system spanning multiple GitHub organizations and repositories.

Use the sandbox to:

- **Develop** against BlackRoad OS APIs without affecting production data
- **Test** integrations and workflows in isolation
- **Validate** package and service integrations before deployment
- **Prototype** new agents, gateways, and services
